# LLM-Based AI/ML Study Assistant Chatbot

## Overview
This project implements an AI/ML study assistant chatbot using an open-source Large Language Model
combined with Retrieval-Augmented Generation (RAG). The chatbot answers questions related to
Artificial Intelligence, Machine Learning, Deep Learning, and NLP by retrieving relevant context
from a custom knowledge base.

## Tech Stack
- Python
- Hugging Face Transformers
- Sentence Transformers
- FAISS
- Google Colab

## Project Structure
- data/raw: Dataset files
- notebooks: Colab notebook for chatbot implementation
- src: Core pipeline scripts

## Status
Project setup completed. RAG pipeline implementation in progress.
